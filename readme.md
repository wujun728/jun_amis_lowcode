# element-shiro

#### 项目介绍
element-shiro轻量级脚手架,基于element ui、springboot、shiro、 mybatis plus、redis、mysql、maven实现.

#### 快速开始

1. 创建数据库,导入resource/sql/element-shiro.sql
2. 导入项目, 修改相关配置, 启动项目
3. 访问http://ip:port, 账号/密码:admin/admin123
